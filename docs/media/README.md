# Media Assets

This directory holds all CHP README media. Assets are generated by
[`build_video.py`](build_video.py) — a Python + PIL + ffmpeg pipeline
that produces both the demo video and the static JPGs in one run.

## Asset inventory

| File | Type | Size | Purpose |
|---|---|---|---|
| `chp-thumbnail.jpg` | JPG 1920×1080 | ~120 KB | README hero / video poster |
| `chp-architecture.jpg` | JPG 1920×1080 | ~170 KB | Six-subsystem architecture diagram |
| `chp-phases.jpg` | JPG 1920×1080 | ~150 KB | Four-phase protocol lifecycle (Propose → Validate → Triangulate → Harden) |
| `chp-social-preview.jpg` | JPG 1280×640 | ~65 KB | GitHub / OpenGraph social preview card |
| `chp-demo.mp4` | MP4 H.264 1920×1080 30fps | ~650 KB | ~30s README demo (8 animated scenes) |
| `chp-demo-3min.mp4` | MP4 H.264 1920×1080 30fps | ~650 KB | Legacy filename kept as a mirror of `chp-demo.mp4` so existing links keep resolving |

## Regenerating

From the repository root:

```bash
python3 docs/media/build_video.py --media-dir docs/media
```

Requirements:

- Python 3.10+
- `pillow` (`pip install pillow`)
- `ffmpeg` (any 4.x or 5.x build with `libx264`)

The script is path-agnostic: pass any target directory with `--media-dir`
and it will drop the JPGs and MP4s there.

## Style

All assets share a single visual system:

- Background: `#0d1117` (GitHub dark)
- Card background: `#161b22`
- Borders: `#30363d`
- Accents: cyan `#58a6ff`, green `#3fb950`, yellow `#d29922`,
  red `#f85149`, purple `#bc8cff`, magenta `#ff63c4`
- Typography: DejaVu Sans (headings), DejaVu Sans Mono (code/labels)
